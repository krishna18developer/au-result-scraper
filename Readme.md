# AU Result Scraper

## Prepare the environment

### Windows
Use Powershell
```
python3 -m venv venv
venv/Scripts/Activate
```

### MacOS / Linux

```
python3 -m venv venv
./venv/Scripts/activate
```

## Install The Packages

```
pip install -r requirements.txt
```

## Using The Script

```
python app.py
```
